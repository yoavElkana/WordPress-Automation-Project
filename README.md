# WordPress Automation Project 
Implementation of E2E automation tests using **Playwright**.

## Features
- **Login Flow**: Positive and negative testing for user authentication.
- **Environment**: Running on a local **Docker** container.
- **Tools**: VS Code, DBeaver (for DB verification), and Git.

## How to run
1. Install dependencies: `npm install`
2. Run tests: `npx playwright test --headed`
